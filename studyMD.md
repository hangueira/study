# 마크다운 정리(\*.md)

## Heading

#### **#** 갯수로 heading 결정

```md
# h1

## h2

### h3

#### h4

##### h5

###### h6
```

---

## Bold

#### \*2개로 감싸면 Bold체(두껍게) 가능

```md
**한경수**
```

---

## italic

#### \* 1개로 or \_ 1개로 감싸면 italic체 가능

(자꾸 변하니까 우선 \* -> \_ 로 치환하여 넣음)

```md
_한경수_
```

---

## 주석처리(strikethrough)

#### ~ 2개로 감싸면 주석처리처럼 변경

```md
~~한경수~~
```

---

## Quote 처리

#### 맨앞에 > 넣으면 됨

```md
> 한경수
```
